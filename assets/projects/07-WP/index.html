Descripción del proyecto

Este proyecto consistió en la creación de un sitio web en WordPress, estructurado en cuatro páginas principales: Inicio, Nosotros, Servicios y Contacto, utilizando el tema Kadence y personalizaciones con Elementor. El diseño incluye un menú desplegable desarrollado con HTML, CSS y JavaScript, y un header y footer personalizados. La interfaz es completamente responsiva, optimizada para dispositivos de escritorio, tablet y móvil, garantizando una experiencia de usuario fluida. Además, se integró un formulario funcional mediante un plugin gratuito, logrando un sitio profesional, funcional y adaptado a las necesidades del cliente.
