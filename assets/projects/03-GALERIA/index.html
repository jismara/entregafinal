Galería Web

#figma #css #html #flexbox #grid

Descripción del proyecto

Galería web creada con un diseño responsivo, maquetada en Figma, utilizando HTML y CSS. La estructura se adapta de manera fluida a diferentes tamaños de pantalla, garantizando una experiencia visual óptima en dispositivos móviles, tablets y computadoras. Incluye elementos interactivos y un diseño moderno que resalta la organización y presentación de imágenes.
